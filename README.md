# mental-health-expert-system

This project is an interactive Mental Health Expert System developed using Python, Experta, and Tkinter. It aims to provide users with initial insights into their mental well-being through a series of structured questions across multiple psychological domains.

⚠️ **Disclaimer:** This system is not a substitute for professional medical advice, diagnosis, or treatment. Always seek the advice of a qualified mental health professional.

---

## Features

- Interactive GUI using Tkinter
- 8-level question structure (e.g., personal, sleep, mood, stress, etc.)
- Knowledge-based decision-making using Experta (Python rule engine)
- Provides:
  - General insights based on answers
  - Suggested coping strategies
  - Recommendation to seek professional help when necessary

---

##  Technologies Used

- Python 3
- [Experta](https://github.com/nilp0inter/experta) – rule-based engine
- Tkinter – graphical interface
